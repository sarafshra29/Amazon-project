📝 Description

A comprehensive data analytics project that analyzes Amazon sales data to generate actionable business insights. This project includes data cleaning, transformation, database design, and an interactive dashboard built using Streamlit.

The dashboard provides deep insights into revenue trends, customer behavior, product performance, and overall business health using advanced visualizations and KPIs.

🚀 Features

📊 Executive Summary Dashboard

Total Revenue, Growth Rate, Active Customers

Average Order Value (AOV)

Top-performing categories

📈 Revenue Trend Analysis

Monthly, Quarterly, Yearly trends

Seasonal patterns & growth rates

Revenue forecasting

🛍️ Product & Category Analysis

Best-selling products

Category-wise performance

Profitability insights

👥 Customer Analytics

Customer segmentation

Repeat vs new customers

Purchase behavior

⚡ Real-time Business Monitoring

Performance vs targets

Alerts for underperformance

🌍 Interactive Visualizations

Plotly charts (bar, line, pie, heatmaps)

Filters (date, category, region)

Dynamic dashboards

🛠️ Tech Stack

Python (Pandas, NumPy)

Streamlit (Dashboard UI)

Plotly (Interactive Visualizations)

PostgreSQL / SQL (Database & Queries)

Jupyter Notebook (Data Cleaning & EDA)
